# Student Management System
Simple Student Management System web application using Spring Boot, Spring MVC, Thymeleaf, Spring Data JPA and MySQL Database.

## Tools and technologies used
* Java 17
* Spring Boot
* Spring MVC
* Spring Data JPA (Hibernate)
* MySQL
* Thymeleaf
* Bootstrap
